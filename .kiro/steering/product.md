# Product Overview

AWS Transcribe Medical proof-of-concept application for demonstrating medical transcription use cases.

## Core Features

- Real-time voice stream transcription
- Audio recording transcription
- Multi-speaker identification and differentiation

## Design Requirements

- Professional, minimal UI using Tailwind CSS and Shadcn
- Simple, intuitive presentation of transcription options
- Full-stack implementation within Next.js (both frontend and backend)
